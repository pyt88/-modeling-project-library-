# Глоссарий терминов

| Термин | Определение |
|--------|-------------|
| **Абонемент** | Выдача книги на дом на определённый срок |
| **Актор** | Внешняя сущность, взаимодействующая с системой (пользователь или другая система) |
| **Артефакт** | Любой создаваемый в процессе разработки продукт (диаграмма, документ, код) |
| **Use Case (прецедент)** | Описание последовательности действий, выполняемых системой для достижения результата, ценного для актора |
| **Ветка (branch)** | Параллельная версия репозитория для изолированной работы |
| **Issue** | Задача, баг или предложение в системе трекинга GitHub |
| **Коммит (commit)** | Зафиксированное изменение в репозитории с комментарием |
| **Репозиторий** | Хранилище файлов проекта с историей изменений |
| **Читатель** | Пользователь системы, берущий книги |
| **Библиотекарь** | Сотрудник, обслуживающий читателей |
| **Администратор** | Пользователь с правами управления системой |
